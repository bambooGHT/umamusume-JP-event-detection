使用[tesseract.js](https://github.com/naptha/tesseract.js) 识别文本

匹配是用很简单的计算两个字符串中相同的字符数跟文本长度的判断,基本匹配出来的都是准确的

不准确的场合 可以自己在correct event中修正事件, 如果有5个以上修正的事件,可以提个issues,我做个整理

由于长度判断在大于原文本的一半之内,如果没有选项时也匹配到了事件,请忽略


没意外会在游戏更新的当天之内更新数据
